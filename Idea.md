Christmas & New Year Themed Website
Project Overview

This website is designed as a festive gift, combining Christmas and New Year themes into a single, interactive experience. The website features five main sections with parallax effects and interactive elements, culminating in a New Year celebration.

Interactive Elements: Snowfall effect, navigation button.
